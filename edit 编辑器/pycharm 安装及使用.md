###[pycharm快捷键](https://www.cnblogs.com/littleseven/p/5599019.html)
- **Alt + F7**/Ctrl + F7文件中查询用法，将usage界面锁定在界面下文
- ctrl + / ：行注释
- Ctrl + Y 删除选定的行
- Ctrl + D 复制选定的区域或行，在下方复制一行
- Ctrl + Backspace 删除到字符开始
- Ctrl + Numpad+/- 展开折叠代码块
- Ctrl + Numpad+ 全部展开
- Ctrl + Numpad- 全部折叠

#####查找/替换类：
- F3 下一个
- Shift + F3 前一个
- Ctrl + R 替换
- Ctrl + Shift + F 全局查找
- Ctrl + Shift + R 全局替换
